# 2-Digit 7-Segment Counter (8051)

This project demonstrates a 2-digit multiplexed 7-segment display using the 8051 microcontroller. The counter increments from 00 to 99 and continuously repeats. The project was developed using Keil uVision and simulated in Proteus.

## Features
- Counts from 00 to 99
- Multiplexed 7-segment display
- Implemented in Embedded C
- Simulated using Proteus
- Developed using Keil uVision

## Folder Structure
- `code/` – Source code
- `keil/` – Keil project files
- `Proteus/` – Proteus simulation files
- `Images/` – Circuit and simulation screenshots

## Software Used
- Keil uVision
- Proteus 8 Professional

## Microcontroller
- AT89S52 / 8051
